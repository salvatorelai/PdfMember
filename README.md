# PdfMember

PdfMember 是一个基于 Python (FastAPI) 和 Vue 3 的 PDF 文档管理与分析平台。

## 功能特性
- PDF 文档上传与管理
- 自动化 AI 分析（生成截图与描述）
- 用户权限与会员管理
- 在线 PDF 阅读器（支持自动缩放与 CJK 字符）

## 项目结构
- `backend/`: FastAPI 后端代码
- `frontend/`: Vue 3 前端代码
- `docs/`: 开发文档与问题记录
- `nginx/`: Nginx 配置文件
- `docker-compose.prod.yml`: 生产环境部署配置
